colide(outro) → {boolean}

Este método encontra-se implementado na superclasse Grafico, assumindo que todas as formas definidas nas suas subclasses são limitadas, de um modo genérico, por um rectângulo imaginário que engloba a totalidade do gráfico. Se este método, tal como se encontra definido, não for o mais apropriado para uma dada subclasse, pode ser sempre redefinido (overrided) nessa mesma subclasse.
